👁️ Padrão Observer — GerenciadorEventosPedido Comportamental

Quando um pedido muda de status, o Sujeito (GerenciadorEventosPedido) itera sobre todos os Observadores registrados e chama atualizar(). Cada observador reage de forma independente.

🏛️ Padrão Facade — FachadaPedido Estrutural

A Facade esconde toda a complexidade. O código cliente (views.py) faz apenas uma chamada e recebe tudo processado.

# SEM Facade (cliente precisaria saber de tudo):
config = ConfiguracaoSistema()
calc = CalculadoraFrete()
calc.definir_estrategia(frete)
pagamento = PagamentoFactory.criar(...)
gerenciador.notificar(...)
# ... validações, cálculos ...
# COM Facade (uma linha):
fachada = FachadaPedido()
resultado = fachada.processar_pedido(dados)
